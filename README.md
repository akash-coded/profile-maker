# user-forum

A demonstrative forum with login, welcome and profile pages.

The site is powered by *PHP 7.2* and employs *Bootstrap4* for visuals.

## Nomenclature

### DB Nomenclature

* Databases - *snake_case*

* Tables - *camelCase*

* Columns - *PascalCase*

* Keys/Constraints
  * Primary Keys - *PK_{Table Name}*
  * Foreign Keys - *FK_{Table Name}*
